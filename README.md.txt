# Currículo Simples em HTML

Este repositório contém um arquivo HTML básico que estrutura um Curriculum Vitae (CV).

## Descrição

O projeto utiliza HTML para apresentar as informações essenciais de um currículo, como dados de contato, objetivo, experiência e estudos.

## Como Usar

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/giannetoo/stackx_cv.git
    ```
2.  **Abra o arquivo:** Navegue até a pasta clonada e abra o arquivo `.html` principal diretamente no seu navegador web.

3.  **Edite:** Modifique o conteúdo do arquivo HTML com suas informações pessoais.

---

Este é um modelo inicial sem estilização CSS.
